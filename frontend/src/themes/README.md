# Теми Huligan Admin

Кожен файл `.js` у цій папці — окрема тема. Після додавання файлу і деплою вона автоматично з'являється у списку в Налаштуваннях.

## Структура файлу теми

```js
// src/themes/my-theme.js

export const name = 'My Theme'   // відображається у дропдауні

export default {
  // Фони
  '--c-bg':        '#...',   // фон сторінки
  '--c-surface':   '#...',   // фон карток, модалок, хедера
  '--c-surface-2': '#...',   // фон інпутів, другорядних кнопок
  '--c-surface-3': '#...',   // фон підтверджень, вкладених панелей

  // Бордери
  '--c-border':       '#...',  // бордери карток
  '--c-border-input': '#...',  // бордери інпутів і селектів

  // Текст
  '--c-text':   '#...',  // основний текст
  '--c-text-2': '#...',  // другорядний текст (лейбли)
  '--c-text-3': '#...',  // приглушений текст (плейсхолдери, підказки)
  '--c-text-4': '#...',  // дуже приглушений текст (id, хінти)

  // Акцент (кнопки, лінки, виділення)
  '--c-accent':        '#...',                  // текст-акцент (лінки, іконки)
  '--c-accent-bg':     '#...',                  // фон основних кнопок
  '--c-accent-hover':  '#...',                  // hover-стан кнопок
  '--c-accent-faint':  'rgb(R G B / 0.12)',     // дуже прозорий акцент (вибраний рядок)
  '--c-accent-border': '#...',                  // рамка при focus та selected-стані
}
```

## Рекомендації

- Темна тема: `--c-bg` темніший за `--c-surface` темніший за `--c-surface-2`
- Світла тема: навпаки — `--c-bg` найсвітліший
- `--c-accent-faint` краще задавати через `rgb(R G B / alpha)` щоб прозорість зберігалась
- Статусні кольори (`completed`, `running`, `failed`, `pending`) і кольори помилок (`red-400`) не залежать від теми — вони залишаються фіксованими
